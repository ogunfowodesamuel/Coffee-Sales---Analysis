# Coffee-Sales---Analysis
The repo contains a project on an exploratory analysis of coffee sales. [Data was gotten from Kaggle]
## Tools
- Microsoft Excel
## Project Overview
This project analyzes the monthly sales performance of a coffe shop.
Explores trends and pattern using interactive dashboard.

## Techniques Used
- Data cleaning
- Pivot Table
- Interactive Visualization
